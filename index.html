Digital Twin Demo


Video demos of Lewistown, Montana, and St. Louis, Missouri, recreated in Unreal Engine 5. This showcases different instances of proposed building projects for the cities which include: existing architecture, new proposal instance switching, and guided tours. Users can change the positions from static cameras to view developments or freely fly around the cities.

Additional techologies include: Realtime OpenWeather API, ArcGIS geography information and CityEngine building creation. Uses Unreal Engine 5 Visual scripting Blueprints System.

3D aim trainer


A Personal 3D aim trainer project created in the Unity game engine. It is a simple game that allows the player to practice aiming in first person shooters. Scripts for the player movement, player actions, and enemy interactions were created using C#.

Reddit Gallery



A convenient and easy way to browse a gallery of images, GIFs, and videos from Reddit.com, utilizing their API. Created using HTML, CSS, and JavaScript. Allows for searching for images by subreddits and sorting images based on how popular the media is on Reddit.
